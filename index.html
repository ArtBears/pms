Poor Man's Spotify - Search, Stream and Download Music

Download .zip Download .tar.gz View on GitHub

pms

  • Search and stream music
  • Download music
  • Works with Python 2.7 and 3
  • Works with Windows and Linux (untested with MacOS)
  • No Python dependencies
  • Requires mplayer

Installation:

Using pip:

sudo pip install Poor-Mans-Spotify

Using git:

git clone https://github.com/np1/pms.git

Manually:

Download zip/tar.gz file and extract

Usage

usage: pms.py query [query ...]

or simply:

pms.py

Screenshot

$ > ls
LICENSE  pms.py  README.md
